## 📊 주요 업체 상세 비교

### 1️⃣ 가비아 (Gabia) - 한국 프로젝트 최고 추천

**장점:**
- ✅ 한국어 고객 지원 (전화, 이메일, 채팅)
- ✅ 한글 도메인 지원 우수
- ✅ .kr 도메인 전문 (한국 도메인 등록원 직접 등록)
- ✅ 원화 결제 가능
- ✅ DNS 관리 인터페이스 한글화
- ✅ 안정적인 서비스 (국내 대표 업체)
- ✅ 다양한 부가 서비스 (이메일, 호스팅 등)

**단점:**
- ❌ 해외 도메인(.com, .net 등) 가격이 다소 비쌈
- ❌ WHOIS 정보 보호 유료 (연간 약 5,000원)

**가격 (2024 기준):**
- `.kr` 도메인: 연간 약 15,000원
- `.com` 도메인: 연간 약 20,000원
- `.net` 도메인: 연간 약 20,000원
- 한글 도메인: 연간 약 25,000원
- WHOIS 보호: 연간 약 5,000원

**추천 대상:**
- 한국 사용자 중심 프로젝트
- 한글 도메인 사용 계획
- .kr 도메인 선호
- 한국어 지원 필요

**웹사이트:** https://www.gabia.com

### 2️⃣ 후이즈 (Whois) - 국내 안정성 추천

**장점:**
- ✅ 한국어 고객 지원 우수
- ✅ .kr 도메인 전문

**단점:**
- ❌ 가비아보다 약간 비쌈
- ❌ WHOIS 보호 유료

### 3️⃣ Cloudflare Registrar - 최저가 추천 ⭐

**장점:**
- ✅ **최저가** (원가 정책 - 마진 없음)
- ✅ WHOIS 정보 보호 **무료**
- ✅ DNS 서비스 **무료** (전 세계 CDN 포함)
- ✅ SSL 인증서 무료 (Let's Encrypt)
- ✅ DDoS 보호 무료
- ✅ 안정적인 글로벌 서비스
- ✅ 갱신 시에도 동일 가격 (프로모션 가격 아님)

**단점:**
- ❌ 한국어 지원 제한적 (영어 위주)
- ❌ 한글 도메인 지원 제한적
- ❌ .kr 도메인 등록 불가 (해외 업체)
- ❌ 신용카드 결제만 가능 (원화 결제 불가)

**가격 (2024 기준):**
- `.com` 도메인: 연간 약 $9.15 (약 12,500원)
- `.net` 도메인: 연간 약 $12.98 (약 17,700원)
- `.org` 도메인: 연간 약 $9.77 (약 13,300원)
- WHOIS 보호: **무료**
- DNS 서비스: **무료**

**추천 대상:**
- 비용 절감 우선
- .com, .net 등 해외 도메인 사용
- DNS 관리 필요
- 보안 기능 중요

**웹사이트:** https://www.cloudflare.com/products/registrar/

**특별 팁:**
- Cloudflare는 도메인 등록 후 DNS를 Cloudflare로 변경하면 추가 보안 기능 제공
- Heroku와 연동 시 DNS 설정이 매우 간편

## 🚀 빠른 구매 가이드

### 가비아로 구매하기 (한국 프로젝트)

1. **회원가입**
   - https://www.gabia.com 접속
   - 회원가입 (이메일 인증 필요)

2. **도메인 검색**
   - 원하는 도메인 입력 (예: `운세담.com`)
   - 사용 가능 여부 확인

3. **도메인 선택 및 결제**
   - 사용 가능한 도메인 선택
   - 등록 기간 선택 (1년 권장)
   - WHOIS 보호 선택 (선택사항)
   - 결제 (신용카드, 계좌이체 등)

4. **DNS 설정**
   - 도메인 관리 → DNS 설정
   - Heroku 연동 준비

### Cloudflare로 구매하기 (비용 절감)

1. **Cloudflare 계정 생성**
   - https://www.cloudflare.com 접속
   - 무료 계정 생성

2. **도메인 등록**
   - Products → Registrar → Register Domains
   - 원하는 도메인 검색 및 등록

3. **DNS 설정**
   - 자동으로 Cloudflare DNS 할당
   - Heroku 연동 시 CNAME 설정

## 💡 추가 팁

### 도메인 선택 팁
1. **짧고 기억하기 쉬운 이름**
   - `운세담.com` ✅
   - `fortune-guide.com` ✅
   - `my-fortune-telling-app-2024.com` ❌

2. **브랜드 일관성**
   - 앱 이름과 도메인 일치
   - 한글/영문 통일

3. **확장자 선택**
   - `.com`: 가장 인지도 높음
   - `.kr`: 한국 프로젝트에 적합
   - `.app`: 앱 프로젝트에 적합 (비쌈)

### WHOIS 보호 필요성
- **필요한 경우:**
  - 개인 정보 보호
  - 스팸 메일 방지
  - 신원 보호

- **Cloudflare는 무료 제공** (추천!)

### DNS 관리
- **가비아/후이즈**: 기본 DNS 제공, 추가 기능 유료
- **Cloudflare**: 고급 DNS 무료 (CDN, 보안 포함)
